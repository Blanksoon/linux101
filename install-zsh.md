# Install zsh and theme oh my themes

### Zsh and oh-my-theme

```
sudo apt-get install -y zsh git
sh -c "$(curl -fsSL https://raw.githubusercontent.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"

which zsh
sudo chsh
/usr/bin/zsh
restart or logout user
```

### Change theme
```
sudo vi ~/.zshrc
ZSH_THEME="cloud"
```
